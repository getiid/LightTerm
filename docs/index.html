开源协作公约

欢迎每一位开发者一起动手，把 AstraShell 做得更稳定、更易用、更有审美。

  • 项目永久开源、永久免费，不做任何形式的功能收费。
  • 官方主线与官方分支不接受任何“收费版/付费分支/付费阉割版”方向。
  • 所有改进优先回馈社区，让每位用户都能平等获得完整能力。
