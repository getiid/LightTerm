## 本次更新（Windows 安装器占用修复）

### 安装器修复
- 新增 NSIS 安装脚本：安装前自动尝试结束 `AstraShell.exe` 进程。
- 降低“覆盖安装提示无法关闭应用”的发生概率。

### 稳定性
- 保留 `v0.2.7` 的 Windows 启动修复（`bindings.node` 架构问题）。

## 安装与升级方法

### Windows
1. 下载 `AstraShell Setup 0.2.8.exe`。
2. 直接运行覆盖安装。
3. 若仍提示占用，请先执行下方命令后再安装。

### macOS（Apple Silicon）
1. 下载 `AstraShell-0.2.8-arm64.dmg`。
2. 手动覆盖安装到 Applications。

## Windows 占用排查命令（管理员 CMD）
`taskkill /F /T /IM AstraShell.exe`
`taskkill /F /T /IM "AstraShell Setup 0.2.6.exe"`
`taskkill /F /T /IM "AstraShell Setup 0.2.7.exe"`
`taskkill /F /T /IM "AstraShell Setup 0.2.8.exe"`
