## 本次更新

### 主机管理体验优化
- 主机卡片交互重构：选择、编辑、连接行为分离，更接近桌面客户端习惯。
- 支持双击卡片直接连接 SSH，会话进入更快。
- 主机卡片状态点支持 SSH 连通性检测，绿色表示可达，红色表示异常。

### 启动与安全流程升级
- 首次启动新增引导：先选择数据库目录，再初始化密钥仓库。
- 后续每次启动默认先解锁密钥仓库，再进入主界面。

### 界面与品牌
- 侧栏品牌区与窗口标题样式优化，整体一致性更高。
- 新增 Windows 安装包发布。

## 安装与使用方法

### Windows
1. 下载 `AstraShell Setup 0.2.6.exe`。
2. 双击安装，默认安装到当前用户目录。
3. 首次打开后按引导完成数据库目录与密钥初始化。

### macOS（Apple Silicon）
1. 下载 `AstraShell-0.2.6-arm64.dmg`。
2. 拖入 Applications 安装。
3. 如果提示来自互联网来源无法打开，执行：
`xattr -dr com.apple.quarantine /Applications/AstraShell.app`

## 升级说明
- 从旧版本升级建议先退出 AstraShell 再安装新版本。
- 若使用本地共享目录数据库，升级后请确认数据库路径配置未变化。
