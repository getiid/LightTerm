## 本次更新（Windows 启动修复）

### 关键修复
- 修复 Windows 安装后启动报错：
`bindings.node is not a valid Win32 application`
- 主进程串口模块改为懒加载，避免原生模块异常导致应用启动即崩溃。
- 打包流程新增清理逻辑，自动剔除错误平台的串口原生产物。

## 安装与使用方法

### Windows
1. 下载 `AstraShell Setup 0.2.7.exe`。
2. 运行安装程序，覆盖安装旧版本。
3. 启动后按引导完成密钥仓库解锁。

### macOS（Apple Silicon）
1. 下载 `AstraShell-0.2.7-arm64.dmg`。
2. 手动覆盖安装到 Applications。

## 升级建议
- 升级前先退出 AstraShell。
- 若安装器提示应用占用，请先执行任务结束后再安装（下方提供命令）。
